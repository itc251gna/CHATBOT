Snippet ενσωμάτωσης

Οι host εφαρμογές ακούνε το hospital-chatbot-termination event και αποφασίζουν τη δική τους επόμενη ροή.

<script
  src="https://10.4.51.232:3443/chatbot-widget.js"
  data-chatbot-api="https://10.4.51.232:3443/api/chat"
  data-chatbot-termination="https://10.4.51.232:3443/api/termination"
  data-chatbot-topics="https://10.4.51.232:3443/api/topics"
  data-title="Τεχνική υποστήριξη"
  data-subtitle="Εσωτερική καθοδήγηση"
></script>

<script>
  window.addEventListener("hospital-chatbot-termination", (event) => {
    console.log(event.detail.signal);
    console.log(event.detail.handoff);
  });
</script>
